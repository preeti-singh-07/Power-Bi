# Sales Performance Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes sales performance across different regions, products, and categories. The report helps identify sales trends, top-performing products, and business growth opportunities.

## Tools Used
- Power BI
- DAX

## Dashboard Features
- Total Sales Analysis
- Profit Analysis
- Regional Performance
- Product Performance
- Interactive Filters and Slicers
